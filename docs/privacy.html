개인정보처리방침

최종 수정일: 2026년 4월 24일  ·  시행일: 2026년 4월 24일

외워줘는 사용자의 학습 데이터를 오직 기기 내부에만 저장합니다. 별도 서버로 개인정보를 전송하지 않습니다.

1. 수집하는 정보

외워줘는 다음 정보를 수집합니다:

  • 학습 데이터 — 카드 세트, 플래시카드 내용, 학습 기록, 복습 스케줄 (기기 내부 SQLite에만 저장)
  • 앱 설정 — 일일 목표, 알림 시간 설정 (기기 내부 파일에만 저장)
  • 광고 식별자 (IDFA/GAID) — 맞춤형 광고 제공 목적, 사용자 동의 시에만 수집

2. 정보의 이용 목적

  • 간격 반복(Spaced Repetition) 알고리즘을 통한 학습 최적화
  • 학습 통계 및 연속 학습 기록 표시
  • 복습 알림 발송
  • 광고 제공 (Google AdMob)

3. 광고 및 제3자 서비스

외워줘는 Google AdMob을 통해 광고를 제공합니다. AdMob은 다음 정보를 수집할 수 있습니다:

  • 광고 식별자 (iOS: IDFA, Android: GAID)
  • 대략적인 위치 정보 (국가/지역 수준)
  • 앱 사용 데이터 (광고 노출/클릭)

iOS 사용자는 앱 최초 실행 시 광고 추적 동의 여부를 선택할 수 있습니다. 거부 시에도 앱의 모든 기능을 정상적으로 이용할 수 있으며, 비맞춤형 광고가 표시됩니다.

Google의 개인정보처리방침: policies.google.com/privacy

4. 데이터 저장 및 보안

  • 모든 학습 데이터는 사용자의 기기에만 저장되며 외부 서버로 전송되지 않습니다.
  • 앱 삭제 시 모든 데이터가 함께 삭제됩니다.
  • 데이터 내보내기 기능을 통해 사용자가 직접 백업할 수 있습니다.

5. 아동 개인정보 보호

외워줘는 만 14세 미만 아동을 대상으로 개인정보를 의도적으로 수집하지 않습니다. 만 14세 미만 사용자는 보호자의 동의 하에 앱을 이용해야 합니다.

6. 사용자의 권리

  • 데이터 삭제 — 앱 삭제 시 모든 데이터가 삭제됩니다.
  • 광고 추적 거부 — iOS: 설정 → 개인정보 보호 → 추적에서 변경 가능
  • 광고 추적 거부 — Android: 설정 → Google → 광고에서 변경 가능

7. 개인정보처리방침 변경

본 방침이 변경되는 경우 앱 업데이트를 통해 공지합니다. 중요한 변경 사항은 앱 내 알림으로 안내합니다.
