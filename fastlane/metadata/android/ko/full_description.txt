외워줘는 과학적으로 검증된 간격 반복(Spaced Repetition) 알고리즘을 사용하여 효율적으로 암기를 도와주는 플래시카드 앱입니다.

■ 주요 기능

• 스마트 복습 스케줄
SuperMemo SM-2 알고리즘 기반으로 각 카드의 난이도와 기억 강도를 분석해 최적의 복습 시점을 자동으로 계산합니다.

• 직관적인 플래시카드
앞면/뒷면을 탭으로 확인하고, 4단계 평가(몰라요·헷갈려요·알아요·완벽해요)로 학습 진도를 기록합니다.

• 오답 노트
틀린 카드만 모아 집중 복습할 수 있어 취약 부분을 빠르게 보완할 수 있습니다.

• 다양한 학습 모드
전체 학습, 오늘의 복습(due), 오답만 복습 세 가지 모드를 지원합니다.

• 프리셋 카드 세트
영어 단어, 한자, JLPT 등 기본 제공 프리셋으로 바로 학습을 시작할 수 있습니다.

• 학습 통계
연속 학습 일수, 누적 학습 카드 수, 정답률을 한눈에 확인하세요.

• 다크 모드
시스템 설정에 따라 자동으로 라이트/다크 모드가 적용됩니다.

• 데이터 백업
JSON 형식으로 데이터를 내보내고 가져올 수 있어 기기 교체 시에도 학습 데이터를 보존할 수 있습니다.

■ 이런 분께 추천합니다
- 외국어(영어, 일본어, 중국어 등) 단어를 외우고 싶은 분
- 자격증·시험 준비 중인 수험생
- 효율적인 암기 방법을 찾는 학생과 직장인
